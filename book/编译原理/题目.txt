# flex
flex 弹性布局
display flex 可以设置任何一个元素

包括 container 和 item 两部分，flex 子元素为 item

容器默认有两个轴，主轴 和 交叉轴

控制主轴排列方式：justify-content
控制交叉轴排列方式：align-item

子元素属性：flex-basic / flex-grow / flex-shrink

defer 和 script
普通script，加载 和 执行 都会阻塞
defer，加载执行会放到dom渲染完毕后
script 加载不阻塞dom渲染，执行会阻塞
